1. Menyalakan server dengan cara mengetikkan berikut di Command Prompt:
> node index.js

2. Jika muncul "App listening on port 3000" maka server sudah menyala

3. Ketikkan localhost:3000 di Search bar browser

4. Login terlebih dahulu melalui Facebook Authentication

5. Ketikkan ID (pokedex number) pokemon yang ingin dicari kelemahannya. 

6. Hasil akan dimunculkan dalam bentuk JSON
